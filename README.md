# QR — Responsive QR Code Generator

**QR** is a fast, modern, and completely **free forever** web app for creating QR codes.  
It’s **ad-free**, **responsive**, and built entirely with **HTML, CSS, and JavaScript** — no frameworks, sign-ups, or dependencies.

Generate, preview, and download your QR codes instantly with **live updates** and **advanced customization** options.

---

## 🌐 What It Does
- Generate QR codes instantly from text or URLs.  
- Customize **colors**, **size**, and **error correction level**.  
- Download high-resolution **PNG** files.  
- Fully **responsive** — works seamlessly on phones, tablets, and desktops.  
- Includes **dark/light mode** and remembers user preferences.  

---

## ⚙️ How to Use
1. Open `index.html` in any modern browser.  
2. Enter your text or link.  
3. Adjust settings (color, size, error level).  
4. Click **Download** to save your QR code.  

✅ No ads.  
✅ No tracking.  
✅ No installation required.

---

## 🧠 Tech Stack
- **HTML5** – Semantic structure  
- **CSS3** – Responsive layout and theming  
- **Vanilla JavaScript (ES6)** – Live preview and user interaction  
- **QRCode.js** – Lightweight QR generation logic  

---

## 🧩 Features
- Live preview updates as you type  
- Dark & light theme with local storage memory  
- Adjustable color pickers with live hex display  
- Output resolution up to **1024px (HD)**  
- Fully accessible with keyboard navigation & focus states
